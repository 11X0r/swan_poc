Swan API Domain Isolation Bypass Proof

This demonstration proves that:

  1. Swan's API accepts cookie-based authentication despite claims it "only works with authorization bearer and no cookies"
  2. Domain isolation between dashboard.swan.io (cookies) and api.swan.io (supposed JWT-only) is failing
  3. The CORS misconfiguration allows exploitation from any arbitrary origin
