Security Vulnerability Details

This page demonstrates a critical CORS vulnerability in Swan's banking API that allows any malicious website to:

  1. Make authenticated requests to Swan's API using a victim's session
  2. Access sensitive user data and perform banking operations
  3. All of this happens silently when a victim visits a malicious webpage

Note: This attack is only possible because of the CORS misconfiguration and would be prevented by Same-Origin Policy if properly configured.
